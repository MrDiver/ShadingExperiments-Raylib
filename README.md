# Configuration

```sh
# Getting the Project ready
git clone https://github.com/MrDiver/ShadingExperiments-Raylib.git
cd ShadingExperiments-Raylib
mkdir build
cmake ..
make
```
## Running the Project
```sh
# Running the project
./dist/rlProgram
```
